  - рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ рЃЊрЃљрЃАрЃљрЃгрЃДрЃўрЃАрЃерЃў рЃўрЃДрЃЮрЃА **рЃЉрЃЮрЃџрЃЮ рЃЏрЃЮрЃЋрЃџрЃћрЃюрЃћрЃЉрЃў**. рЃ»рЃћрЃа рЃљрЃЦрЃбрЃўрЃБрЃарЃў рЃАрЃљрЃЏрЃАрЃљрЃ«рЃБрЃарЃў, рЃерЃћрЃЏрЃЊрЃћрЃњ рЃгрЃўрЃюрЃћрЃЉрЃў, рЃерЃћрЃЏрЃЊрЃћрЃњ рЃњрЃљрЃюрЃљрЃЌрЃџрЃћрЃЉрЃљ...
  - рЃЦрЃарЃЮрЃюрЃЮрЃџрЃЮрЃњрЃўрЃўрЃА рЃерЃћрЃЊрЃњрЃћрЃюрЃўрЃАрЃљрЃА рЃерЃћрЃћрЃфрЃљрЃЊрЃћрЃЌ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў рЃцрЃљрЃЦрЃбрЃћрЃЉрЃў рЃўрЃДрЃЮрЃА рЃюрЃљрЃЎрЃџрЃћрЃЉрЃљрЃЊ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃќрЃћ рЃќрЃћрЃЋрЃўрЃЌ.
  - рЃўрЃЊрЃћрЃљрЃџрЃБрЃа рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃарЃћрЃЎрЃарЃБрЃбрЃћрЃарЃЏрЃљ рЃюрЃљрЃ«рЃћрЃЋрЃљрЃа рЃњрЃЋрЃћрЃарЃЊрЃА рЃарЃЮрЃЏ рЃњрЃљрЃЊрЃљрЃљрЃЋрЃџрЃЮрЃА рЃЌрЃЋрЃљрЃџрЃў, рЃАрЃљрЃАрЃБрЃарЃЋрЃћрЃџрЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃЏрЃљрЃўрЃюрЃф рЃБрЃюрЃЊрЃљ рЃЏрЃўрЃўрЃдрЃЮрЃА.  

## 8. рЃ░рЃЮрЃЉрЃў
  - рЃљрЃќрЃарЃЌрЃљ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљрЃЮрЃЉрЃљрЃљ рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃЌрЃБ рЃљрЃарЃљ рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃБрЃџрЃў рЃЎрЃљрЃюрЃЊрЃўрЃЊрЃљрЃбрЃўрЃА рЃ░рЃЮрЃЉрЃў рЃљрЃю рЃАрЃ«рЃЋрЃљ рЃърЃўрЃарЃљрЃЊрЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ.
  - рЃќрЃЮрЃњрЃўрЃА рЃљрЃќрЃарЃўрЃЌ рЃ░рЃЮрЃЉрЃўрЃА рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃљ рЃљрЃа рЃљрЃарЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮ, рЃарЃљрЃЊрЃњрЃљрЃю рЃърЃарЃЮрЃцрЃћрЃАрЃўрЃљрЃАрЃЌрЃљрЃю рЃљрЃарЃљрЃюрЃљрЃўрЃарЃў рЃЎрЃљрЃЋрЃерЃўрЃарЃў рЃљрЃа рЃљрЃЦрЃЋрЃА.
  - рЃќрЃЮрЃњрЃў рЃЌрЃЋрЃџрЃўрЃА, рЃарЃЮрЃЏ рЃЏрЃфрЃўрЃарЃћрЃЊрЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ рЃљрЃЊрЃљрЃЏрЃўрЃљрЃюрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ рЃАрЃљрЃАрЃљрЃарЃњрЃћрЃЉрЃџрЃЮ рЃљрЃарЃўрЃА рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃўрЃџрЃћрЃЉрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃўрЃАрЃљрЃА.
  - рЃ░рЃЮрЃЉрЃўрЃА рЃЕрЃљрЃгрЃћрЃарЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў, рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ **рЃфрЃЮрЃбрЃљ рЃАрЃўрЃЋрЃарЃфрЃћ рЃЊрЃљ рЃЊрЃљрЃгрЃћрЃарЃЮрЃЌ рЃЉрЃЮрЃџрЃЮрЃерЃў**.

**[Ргє рЃБрЃЎрЃљрЃю рЃАрЃљрЃарЃЕрЃћрЃЋрЃерЃў](#рЃАрЃљрЃарЃЕрЃћрЃЋрЃў)**
